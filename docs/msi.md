# MSI

## Software da mantenere

Su MSI mantenere normalmente:

- MSI Center
- driver chipset
- driver audio
- driver Intel / Wi‑Fi / Bluetooth
- driver NVIDIA
- utility necessarie per ventole, profili e MUX

Rimuovere solo bloat evidente, trial, launcher o app non necessarie.

## Modalità prestazioni

Nomi e opzioni cambiano in base al modello, ma la logica è questa:

### DAILY MODE

- Modalità GPU: Hybrid / MSHybrid / Optimized, se disponibile.
- Profilo: Balanced / Performance moderato.

### SHOW MODE

- Modalità GPU: Discrete / dGPU only / MUX su NVIDIA.
- Profilo: Extreme Performance / Turbo equivalente.
- Cooler Boost solo se serve e dopo test acustico/termico.

## Note

Su alcuni MSI il cambio MUX richiede riavvio. Farlo sempre prima di aprire software live.
