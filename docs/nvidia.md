# NVIDIA

## Driver

Per workstation live usare preferibilmente:

- NVIDIA Studio Driver

Game Ready può andare bene, ma Studio è in genere più prudente per software creativi, rendering e produzione.

## Installazione

Quando si aggiorna il driver:

1. NVIDIA App.
2. Driver Studio.
3. Installazione personalizzata.
4. Installazione pulita.
5. Riavvio.

Non aggiornare driver NVIDIA subito prima di uno spettacolo.

## Pannello di controllo NVIDIA

Impostazioni globali consigliate come base:

- GPU preferita: NVIDIA ad alte prestazioni.
- Power management: Preferisci prestazioni massime, almeno per le app live.
- Low Latency Mode: On.
- Vertical Sync: Off globalmente, poi decidere dentro i software/progetto.
- Shader cache: On / Default.
- G‑Sync: off per show/proiettori, salvo casi specifici.

## Impostazioni per app

Creare profili specifici per:

- TouchDesigner
- MadMapper
- Resolume
- Unreal
- Blender
- Notch

Per queste app: preferire prestazioni massime e GPU NVIDIA.

## HAGS, VRR, Game Mode

Queste opzioni non hanno una risposta universale.

- HAGS: testare ON e OFF con progetti reali.
- VRR: utile nel daily, da valutare per show/proiettori.
- Game Mode: generalmente lasciabile attivo, salvo problemi specifici.

Annotare nel repo quale combinazione funziona meglio per ogni macchina/progetto.
