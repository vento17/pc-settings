# Checklist SHOW MODE

Da fare prima di prove, show, rendering pesanti o media server.

## Power / GPU

- [ ] Alimentatore collegato.
- [ ] Batteria presente e carica.
- [ ] Modalità GPU: Ultimate / Discrete / dGPU only.
- [ ] Riavvio fatto dopo cambio MUX, se richiesto.
- [ ] Profilo: Turbo / Extreme Performance.

## Windows

- [ ] Windows Update in pausa.
- [ ] Sospensione: Mai.
- [ ] Notifiche / Non disturbare attivo.
- [ ] OneDrive/cloud sync in pausa o chiuso.
- [ ] Browser e software inutili chiusi.

## Rete

- [ ] Ethernet collegata.
- [ ] IP/subnet corretti.
- [ ] Wi‑Fi disattivato se crea routing ambiguo.
- [ ] Art-Net/sACN/OSC testati.

## USB / periferiche

- [ ] SSD show collegato.
- [ ] DMX USB / MIDI / audio / capture collegati.
- [ ] Hub alimentati, se necessari.
- [ ] Nessun hotplug inutile durante lo show.

## Video

- [ ] Proiettori/monitor riconosciuti.
- [ ] Risoluzione corretta.
- [ ] Refresh coerente, normalmente 60 Hz per proiettori.
- [ ] Test fullscreen/output.

## Software

- [ ] Aprire TouchDesigner/MadMapper dopo avere collegato le periferiche.
- [ ] Verificare FPS.
- [ ] Verificare media path.
- [ ] Verificare audio/DMX/OSC/MIDI.
- [ ] Backup progetto disponibile.
