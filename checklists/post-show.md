# Checklist POST SHOW

Da fare dopo lo spettacolo o dopo una giornata di prove.

- [ ] Salvare progetto finale.
- [ ] Copiare media/progetto su backup.
- [ ] Annotare problemi incontrati.
- [ ] Annotare driver/versioni se qualcosa è cambiato.
- [ ] Verificare spazio libero su SSD.
- [ ] Chiudere software live.
- [ ] Spegnere periferiche in ordine.
- [ ] Riattivare aggiornamenti solo quando c'è tempo per testare.
- [ ] Se la configurazione è stabile, creare nuovo punto di ripristino o immagine disco.
